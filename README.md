# Change Driven Meta Model System (MMS)
 
Welcome to the Group 5 MMS repository for the MSc Systems Engineering project at Loughborough University.
 
This repository is used to manage the development of our change-driven Meta Model System (MMS), supporting configuration control, engineering change management, and integration between SysML and external specialist models including MATLAB, Relyence, FreeCAD, and Excel.
 
GitHub is being used as the project Configuration Control System (CCS), providing traceability through commits, branches, pull requests, and baseline management throughout the integration sprint.
